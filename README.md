# 🏦 Loan Portfolio Risk Analytics

A BigQuery SQL project focused on analyzing loan approval patterns, customer demographics, income behavior, and credit risk indicators.

[![View Presentation](https://img.shields.io/badge/📄_View_Project_Deck-PDF-red?style=for-the-badge)](./Loan_Risk_Portfolio_Presentation.pdf)

## 📌 Project Overview

This project combines customer, loan, deposit, and credit card datasets to uncover key factors influencing loan approvals and customer risk profiles.

## 🔍 Key Analyses

* Loan Approval Rate Analysis
* Approval Trends by Gender & Education
* Property Area Performance
* Credit History Impact on Approvals
* Income Ranking & Segmentation
* Deposit-Based Customer Segmentation
* Customer Risk Classification

## 💡 Key Insights

* Credit history emerged as the strongest approval driver.
* Higher income did not always guarantee loan approval.
* Approval patterns varied across customer segments.
* Deposit behavior helped identify lower-risk customers.

## 🛠 Skills Demonstrated

* Google BigQuery
* SQL Joins
* Aggregations
* CASE WHEN
* Window Functions (RANK, NTILE)
* Customer Segmentation
* Risk Analytics

## 📂 Dataset

The analysis was performed using four related tables:

* Customers
* Loans
* DepositCustomers
* HighCreditBalanceCustomers
